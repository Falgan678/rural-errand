# 小程序提审前功能测试清单

> 备案等待期间，按本清单跑一遍所有功能，确保提审时审核员不会卡在任何一个 bug 上。
> **每测一项打 ✓ 或 ✗，✗ 的告诉我我修。**

---

## 🧪 测试环境准备

1. 打开 **微信开发者工具**
2. 顶部点 **编译** （Ctrl+B）
3. 打开 **调试器 → Console** 标签（看报错日志）
4. 右上角 **详情 → 本地设置** 确认勾选：
   - ☑ 不校验合法域名、TLS、HTTPS 证书
   - ☑ ES6 转 ES5
   - ☑ 增强编译
5. 模拟器选 **iPhone 12/13/14**（最常见尺寸）
