# 腾讯云短信服务配置指南

## 1. 开通腾讯云短信服务

### 1.1 注册腾讯云账号
访问 [腾讯云官网](https://cloud.tencent.com/) 注册账号并完成实名认证。

### 1.2 开通短信服务
1. 登录 [腾讯云短信控制台](https://console.cloud.tencent.com/smsv2)
2. 点击"开始使用"开通短信服务
3. 完成企业认证（个人用户也可以使用，但有限制）

## 2. 配置短信服务

### 2.1 创建签名
1. 在短信控制台左侧菜单选择"国内短信" -> "签名管理"
2. 点击"创建签名"
3. 填写签名信息：
   - 签名类型：选择"App"或"网站"
   - 签名用途：选择"自用"
   - 签名内容：填写"农村跑腿"（或您的应用名称）
   - 上传相关证明材料
4. 提交审核，等待审核通过（通常1-2个工作日）

### 2.2 创建短信模板
1. 在短信控制台左侧菜单选择"国内短信" -> "正文模板管理"
2. 点击"创建正文模板"
3. 填写模板信息：
   - 模板名称：验证码短信
   - 短信类型：验证码
   - 短信内容：`您的验证码是{1}，{2}分钟内有效，请勿泄露给他人。`
4. 提交审核，等待审核通过
5. 审核通过后，记录模板ID（例如：1234567）

### 2.3 创建应用
1. 在短信控制台左侧菜单选择"应用管理" -> "应用列表"
2. 点击"创建应用"
3. 填写应用名称：农村跑腿
4. 创建成功后，记录 SDK AppID（例如：1400123456）

### 2.4 获取API密钥
1. 访问 [API密钥管理](https://console.cloud.tencent.com/cam/capi)
2. 点击"新建密钥"
3. 记录 SecretId 和 SecretKey（请妥善保管，不要泄露）

## 3. 配置环境变量

在部署时，需要配置以下环境变量：

```bash
# 腾讯云API密钥
TENCENT_SECRET_ID=your_secret_id_here
TENCENT_SECRET_KEY=your_secret_key_here

# 短信应用配置
SMS_SDK_APP_ID=1400123456
SMS_SIGN_NAME=农村跑腿
SMS_TEMPLATE_ID=1234567
```

### 3.1 本地开发配置
创建 `.env` 文件（不要提交到Git）：

```bash
TENCENT_SECRET_ID=AKIDxxxxxxxxxxxxxxxxxxxxx
TENCENT_SECRET_KEY=xxxxxxxxxxxxxxxxxxxxxxxx
SMS_SDK_APP_ID=1400123456
SMS_SIGN_NAME=农村跑腿
SMS_TEMPLATE_ID=1234567
```

### 3.2 生产环境配置
在服务器或容器平台配置环境变量，或通过配置管理工具注入。

## 4. 短信模板参数说明

当前代码中使用的模板参数：
- `{1}`: 验证码（6位数字）
- `{2}`: 有效期（固定为5分钟）

示例短信内容：
```
【农村跑腿】您的验证码是123456，5分钟内有效，请勿泄露给他人。
```

## 6. 开发模式

如果未配置腾讯云短信服务（环境变量为空），系统会自动进入**模拟模式**：
- 不会真实发送短信
- 验证码会打印在服务器控制台
- 适合开发和测试环境使用

## 7. 常见问题

### 7.1 短信发送失败
- 检查签名和模板是否审核通过
- 检查手机号格式是否正确
- 检查账户余额是否充足
- 查看错误日志获取详细错误信息

### 7.2 验证码收不到
- 检查手机号是否正确
- 检查是否被运营商拦截（可能被识别为垃圾短信）
- 检查短信发送频率限制（同一手机号1分钟内只能发送1条）

### 7.3 安全建议
- 不要将API密钥提交到代码仓库
- 定期更换API密钥
- 设置短信发送频率限制
- 添加图形验证码防止恶意刷短信
- 生产环境使用Redis存储验证码，设置过期时间
